**TaskFlow Analytics | Gestão de Tarefas, SLA e Produtividade**

**Sobre o projeto**
O TaskFlow Analytics é um projeto fictício desenvolvido para demonstrar, na prática, a construção de uma solução analítica voltada ao acompanhamento de tarefas, controle de SLA e monitoramento da produtividade operacional. A proposta foi estruturar um fluxo completo de dados, conectando Google Sheets, SQL e Supabase para transformar registros operacionais em uma base analítica organizada, com indicadores, tabelas dinâmicas e dashboard gerencial.

**O que este projeto demonstra Este projeto evidencia competências importantes para atuação em análise de dados e funções analíticas de negócio, como:**
- estruturação e organização de dados
- pensamento analítico aplicado a processos operacionais;
- modelagem de dados relacional;
- uso de SQL para criação, transformação e consulta de dados;
- construção de indicadores de desempenho;
- análise de SLA, atrasos e tempo médio de conclusão;
- desenvolvimento de dashboard com foco em leitura executiva;
- conexão entre operação, banco de dados e tomada de decisão.

**Tecnologias e recursos utilizados**
-Google Sheets
- Supabase
- SQL
- CSV
- Tabelas Dinâmicas
- Dashboard
- Modelagem de Dados
- Análise de Dados

**Estrutura da solução**
A solução foi organizada em três frentes principais:
- camada operacional, com registro e organização das tarefas no Google Sheets;
- camada relacional, com modelagem e armazenamento no Supabase;
- camada analítica, com base consolidada, indicadores e dashboard.
Essa arquitetura permitiu transformar uma rotina operacional em uma solução estruturada para acompanhamento gerencial.

**Principais análises desenvolvidas Entre as análises construídas no projeto, estão:**
- total de tarefas;
- tarefas abertas, concluídas e atrasadas;
- percentual dentro e fora do SLA;
- tempo médio de conclusão;
- distribuição de tarefas por equipe, colaborador, prioridade, tipo e solicitante;
- análises temporais de abertura e conclusão.

**Valor profissional do projeto**
Mais do que um exercício técnico, este projeto foi desenvolvido para demonstrar capacidade de:
- entender um problema de negócio;
- estruturar dados de forma lógica;
- construir uma solução analítica aplicável;
- transformar dados em informações úteis para gestão.
O TaskFlow Analytics representa, portanto, uma entrega de portfólio voltada à demonstração prática de habilidades em análise de dados, raciocínio analítico, organização de processos e construção de soluções orientadas a indicadores.
